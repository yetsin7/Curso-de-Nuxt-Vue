# Curso de Nuxt Vue — Ruta completa desde HTML hasta una app real

Este repositorio contiene una sola ruta de estudio, dividida en cuatro cursos
independientes pero conectados:

1. `01- HTML`
2. `02- CSS`
3. `03- Java Script`
4. `04- Nuxt Vue`

La idea no es mezclar veinte tecnologias. La idea es dominar la ruta correcta:

- primero estructura con HTML;
- luego presentacion con CSS;
- despues logica con JavaScript;
- finalmente aplicaciones reales con Vue y Nuxt.

---

## Como estudiar este repositorio

Sigue este orden exacto:

1. Abre el `README.md` de `01- HTML`
2. Recorre sus modulos en orden
3. Pasa a `02- CSS`
4. Luego estudia `03- Java Script`
5. Termina con `04- Nuxt Vue`

Cada carpeta fue pensada para poder estudiarse por si sola, pero juntas forman
una ruta coherente hacia Nuxt Vue.

---

## Regla de estudio

- escribe el codigo a mano;
- ejecuta o abre cada ejemplo;
- no avances por ansiedad;
- solo pasa al siguiente bloque cuando puedas explicar el actual;
- si un error aparece, leelo y entendelo antes de pedir ayuda.

---

## Objetivo final

Al terminar la ruta deberias poder:

- construir paginas reales con HTML;
- darles estilo profesional con CSS;
- agregar comportamiento con JavaScript;
- crear interfaces por componentes con Vue;
- construir aplicaciones web modernas con Nuxt.

---

## Nota sobre Git

Este repositorio debe vivir como un solo proyecto:

- sin repositorios git anidados en las carpetas internas;
- con una sola raiz apuntando a `https://github.com/yetsin7/curso-nuxt-vue.git`.
