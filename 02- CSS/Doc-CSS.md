# Documentacion de CSS

CSS significa Cascading Style Sheets. Sirve para darle diseno, colores,
tamanos, posiciones y estilo visual a una pagina web.

## Documentacion recomendada

- [MDN CSS Docs](https://developer.mozilla.org/en-US/docs/Web/CSS)
- [W3C CSS](https://www.w3.org/Style/CSS/)

## Temas principales que debes estudiar

1. Selectores CSS
2. Colores y tipografia
3. Box model
4. Margin, padding y border
5. Display y position
6. Flexbox
7. Grid
8. Responsive design
9. Media queries
10. Animaciones y transiciones
11. Variables CSS
12. Pseudo-clases y pseudo-elementos

## Orden recomendado

1. [Fundamentos de CSS](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Styling_basics)
2. [Selectores CSS](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_selectors)
3. [Box model](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Styling_basics/Box_model)
4. [Flexbox](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/CSS_layout/Flexbox)
5. [Grid](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/CSS_layout/Grids)
6. [Responsive design](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/CSS_layout/Responsive_Design)

## Ejemplo basico

```css
body {
  font-family: Arial, sans-serif;
  background-color: #f4f4f4;
  color: #222;
}

h1 {
  color: #0066cc;
}

p {
  font-size: 18px;
}
```
