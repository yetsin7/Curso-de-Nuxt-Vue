# Curso de CSS — De cero a interfaces limpias y responsivas

Este curso esta pensado para aprender CSS como una herramienta seria de
presentacion y layout, no como un conjunto de trucos.

---

## Que aprenderas

- sintaxis de CSS;
- box model;
- colores, fondos y tipografia;
- espaciado correcto;
- flexbox;
- grid;
- responsive design;
- como vestir una pagina HTML real.

---

## Modulos del curso

1. `01-introduccion-y-sintaxis`
2. `02-box-model-y-espaciado`
3. `03-tipografia-colores-y-fondos`
4. `04-flexbox`
5. `05-grid-y-responsive`
6. `06-proyecto-final-css`

---

## Resultado esperado

Al terminar deberias poder convertir una pagina HTML cruda en una interfaz:

- clara;
- ordenada;
- legible;
- responsiva;
- lista para integrarse con JavaScript o Vue.
