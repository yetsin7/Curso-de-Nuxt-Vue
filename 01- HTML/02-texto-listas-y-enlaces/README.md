# Modulo 02 — Texto, listas y enlaces

## Objetivo

Aprender a organizar contenido escrito de manera clara y navegable.

## Conceptos

- `h1` a `h6`;
- `p`;
- `strong`, `em`;
- `ul`, `ol`, `li`;
- `a`;
- jerarquia de contenido.

## Explicacion simple

Una pagina no es solo "poner texto". Debe tener orden.

La jerarquia importa porque:

- el usuario entiende mejor;
- el navegador entiende mejor;
- luego SEO tambien mejora.

## Ejercicio

Construye una pagina que tenga:

- un titulo principal;
- dos secciones con `h2`;
- una lista de pasos;
- una lista de beneficios;
- un enlace a otra pagina o recurso.

## Mini proyecto

Crea una pagina tipo \"guia rapida\" para aprender una habilidad simple.

## Debes dominar antes de avanzar

- usar encabezados en orden;
- crear listas ordenadas y no ordenadas;
- enlazar contenido sin romper la estructura.
