# Curso de HTML — De cero a una base profesional de estructura web

Este curso ya no se queda en una introduccion corta. La idea es que HTML quede
lo bastante trabajado como para sostener paginas reales, landing pages,
formularios, catalogos y la base de una app moderna.

## Modulos del curso

1. `01-introduccion-y-estructura`
2. `02-texto-listas-y-enlaces`
3. `03-imagenes-tablas-y-embebidos`
4. `04-formularios-completos`
5. `05-html-semantico-y-accesibilidad`
6. `06-proyecto-final-html`
7. `07-seo-y-metadatos-html`
8. `08-multimedia-y-embed-avanzado`
9. `09-estructuras-de-blog-y-noticias`
10. `10-paginas-de-venta-y-landing`
11. `11-componentes-estaticos-de-ecommerce`
12. `12-proyecto-final-html-pro`

## Resultado esperado

Al terminar deberias poder estructurar:

- paginas informativas;
- landings;
- catalogos;
- paginas de producto;
- formularios;
- una mini tienda estatica base.
