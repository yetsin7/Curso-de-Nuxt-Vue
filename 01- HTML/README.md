# Curso de HTML — De cero a paginas bien estructuradas

HTML es la base de todo lo que haras despues. Si esta base queda floja, CSS se
vuelve confuso y JavaScript se vuelve desordenado.

Este curso esta pensado para alguien que nunca ha trabajado en serio con HTML y
quiere aprenderlo con enfoque real.

---

## Que aprenderas

- como se construye un documento HTML;
- como organizar texto y secciones;
- como usar listas, enlaces e imagenes;
- como crear formularios utiles;
- como escribir HTML semantico;
- como estructurar una landing page y una pagina de producto.

---

## Modulos del curso

1. `01-introduccion-y-estructura`
2. `02-texto-listas-y-enlaces`
3. `03-imagenes-tablas-y-embebidos`
4. `04-formularios-completos`
5. `05-html-semantico-y-accesibilidad`
6. `06-proyecto-final-html`

---

## Resultado esperado

Al terminar este curso deberias poder crear sin copiar:

- una pagina informativa;
- una landing page simple;
- una pagina de producto;
- una estructura HTML lista para recibir CSS y JavaScript.
