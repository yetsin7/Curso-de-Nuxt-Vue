# Modulo 06 — Proyecto final HTML

## Objetivo

Construir una pagina completa de producto o landing page usando solo HTML.

## Debe incluir

- cabecera;
- hero;
- beneficios;
- prueba social;
- seccion FAQ;
- formulario o CTA;
- footer.

## Lo que se evalua

- estructura;
- jerarquia;
- semantica;
- claridad comercial;
- orden del documento.

## Resultado

Si terminas este proyecto bien, ya tienes una base real para pasar a CSS.
