# Documentacion de HTML

HTML significa HyperText Markup Language. Es el lenguaje base para crear la
estructura de una pagina web.

## Documentacion recomendada

- [MDN HTML Docs](https://developer.mozilla.org/en-US/docs/Web/HTML)
- [WHATWG HTML Living Standard](https://html.spec.whatwg.org/)

## Temas principales que debes estudiar

1. Estructura basica de un documento HTML
2. Etiquetas principales
3. Encabezados y parrafos
4. Enlaces e imagenes
5. Listas y tablas
6. Formularios e inputs
7. HTML semantico
8. Multimedia
9. Accesibilidad basica
10. SEO basico con HTML

## Orden recomendado

1. [Estructura basica](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Structuring_content/Basic_HTML_syntax)
2. [Elementos HTML](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements)
3. [Formularios](https://developer.mozilla.org/en-US/docs/Learn_web_development/Extensions/Forms)
4. [Semantica](https://developer.mozilla.org/en-US/docs/Glossary/Semantics)

## Ejemplo basico

```html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mi primera pagina</title>
</head>
<body>
  <h1>Hola mundo</h1>
  <p>Esta es mi primera pagina con HTML.</p>
</body>
</html>
```
