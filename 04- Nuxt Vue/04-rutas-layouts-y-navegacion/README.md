# Modulo 04 — Rutas, layouts y navegacion

## Objetivo

Dar el salto de Vue a Nuxt entendiendo como se organizan paginas reales.

## Conceptos

- `pages`;
- rutas automáticas;
- layouts;
- navegacion;
- rutas dinamicas.

## Explicacion simple

Nuxt te da una arquitectura:

- `pages` crea URLs;
- `layouts` evita repetir estructura;
- `components` guarda piezas reutilizables.

## Ejercicio

Diseña una mini app con:

- home;
- pagina de productos;
- detalle de producto;
- contacto.

## Debes dominar antes de avanzar

- saber donde va cada archivo;
- entender la diferencia entre pagina y componente.
