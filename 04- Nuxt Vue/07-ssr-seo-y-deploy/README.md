# Modulo 07 — SSR, SEO y deploy

## Objetivo

Entender por que Nuxt es tan util para proyectos que deben ser visibles,
rapidos y organizados.

## Conceptos

- SSR;
- SEO basico;
- metadatos;
- estructura de paginas indexables;
- despliegue.

## Explicacion simple

SSR ayuda porque el servidor puede entregar HTML listo para ser interpretado por
navegadores y buscadores.

Eso aporta especialmente en:

- home;
- paginas de producto;
- categorias;
- contenido publico.

## Debes dominar antes de avanzar

- explicar por que Nuxt aporta a SEO;
- saber que paginas de tu app deberian pensarse para SSR.
