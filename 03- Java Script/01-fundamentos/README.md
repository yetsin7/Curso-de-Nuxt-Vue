# Modulo 01 — Fundamentos

## Objetivo del modulo

Entender que es JavaScript, como se ejecuta y por que variables, scope y salida
en consola son tan importantes desde el principio.

## Que aprenderas

- que es JavaScript;
- diferencia entre navegador y Node como entorno de ejecucion;
- como mostrar resultados en consola;
- variables basicas;
- primeras reglas de sintaxis.

## Ejercicios recomendados

1. Ejecuta cada archivo del modulo.
2. Cambia los valores y vuelve a ejecutarlo.
3. Explica con tus palabras que hace cada linea.
4. Rompe un ejemplo a proposito y mira el error.
