# Modulo 03 — Operadores

## Objetivo del modulo

Aprender a combinar, comparar y transformar datos usando operadores basicos del
lenguaje.

## Que aprenderas

- operadores aritmeticos;
- operadores de comparacion;
- operadores logicos;
- algunos operadores utiles modernos.

## Relacion con problemas reales

Con operadores puedes calcular subtotales, validar rangos, combinar condiciones
y revisar si un usuario puede o no hacer algo.
