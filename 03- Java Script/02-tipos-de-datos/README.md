# Modulo 02 — Tipos de datos

## Objetivo del modulo

Aprender los tipos basicos que JavaScript maneja y como pensar mejor los datos
antes de operar con ellos.

## Que aprenderas

- strings;
- numbers;
- booleans;
- `null` y `undefined`;
- diferencias entre tipos y sus usos comunes.

## Importancia real

Si no entiendes bien los datos, luego se rompen validaciones, condiciones,
calculos, formularios y consumo de APIs.
