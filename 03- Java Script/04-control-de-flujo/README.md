# Modulo 04 — Control de flujo

## Objetivo del modulo

Entender como hacer que el programa tome decisiones y repita tareas cuando sea
necesario.

## Que aprenderas

- `if` y `else`;
- `for`;
- `while`;
- `break` y `continue`.

## Casos reales

Esto se usa cuando quieres mostrar mensajes distintos, recorrer productos,
aplicar descuentos y validar datos.
