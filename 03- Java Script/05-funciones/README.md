# Modulo 05 — Funciones

## Objetivo del modulo

Aprender a encapsular logica para no repetir codigo y empezar a pensar de forma
mas modular.

## Que aprenderas

- funciones declaradas;
- arrow functions;
- parametros;
- retorno;
- closures a nivel introductorio.

## Por que importa

Sin funciones, el codigo crece mal. Con funciones puedes construir piezas como
`calcularTotal()`, `validarFormulario()` y `filtrarProductos()`.
