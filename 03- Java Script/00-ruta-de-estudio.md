# Ruta de estudio — Java Script para frontend real

## Si eres principiante

No intentes abrir todos los archivos a la vez.
Haz esto:

1. lee el `README.md` principal del curso;
2. entra al modulo 01;
3. ejecuta los ejemplos;
4. haz los ejercicios;
5. no avances hasta entender que hace el codigo.

## Prioridad real

Lo mas importante no es POO ni tooling.
Lo mas importante para llegar a Vue y Nuxt es dominar:

- variables;
- condiciones;
- funciones;
- arrays;
- objetos;
- DOM;
- eventos;
- asincronia;
- fetch.

## Meta del curso

Que JavaScript deje de ser sintaxis rara y empiece a sentirse como una
herramienta para resolver problemas de interfaz y datos.
