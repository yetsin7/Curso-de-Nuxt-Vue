# Modulo 06 — Arrays

## Objetivo del modulo

Entender como trabajar con colecciones de datos, algo esencial para cualquier
interfaz moderna.

## Que aprenderas

- arrays basicos;
- metodos funcionales;
- busqueda y ordenamiento;
- desestructuracion.

## Casos reales

Los arrays aparecen en listas de productos, categorias, pedidos, comentarios y
resultados de busqueda.
