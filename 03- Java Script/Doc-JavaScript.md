# Documentacion de JavaScript

JavaScript es el lenguaje principal del navegador. Sirve para crear
interactividad, manejar datos, eventos, formularios, APIs y aplicaciones web.

## Documentacion recomendada

- [MDN JavaScript Docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
- [ECMAScript / ECMA-262](https://ecma-international.org/publications-and-standards/standards/ecma-262/)

## Temas principales que debes estudiar

1. Variables y tipos de datos
2. Operadores y condicionales
3. Bucles y funciones
4. Arrays y objetos
5. DOM y eventos
6. Formularios
7. Fetch API
8. Promesas
9. Async / await
10. Modulos, clases y errores

## Orden recomendado

1. [Fundamentos](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Scripting)
2. [Referencia completa](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference)
3. [Funciones](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Functions)
4. [Objetos](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Working_with_objects)
5. [Promesas](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise)
6. [Async functions](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/async_function)
7. [Fetch API](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API)

## Ejemplo basico

```js
const nombre = "Mauricio";

function saludar(usuario) {
  return `Hola, ${usuario}`;
}

console.log(saludar(nombre));
```
